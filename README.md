# Gram Panchayat Hiwargaon Pawasa

Modern React + Vite demo website.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Free hosting

Use Vercel or Cloudflare Pages:
- Build command: `npm run build`
- Output directory: `dist`
